# Stress Card Game
A favourite childhood game of mine, now playable on web. 

## Technical
- Client: Typescript + React
- Server: Express + Node.js + Websockets
- Deployment: AWS EC2
- DB for recent matches panel: AWS API Gateway, Lambda (GET/POST), Dynamodb
